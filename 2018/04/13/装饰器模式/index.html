何谓装饰器模式

装饰模式(Decorator Pattern) ：动态地给一个对象增加一些额外的职责(Responsibility)，就增加对象功能来说，装饰模式比生成子类实现更为灵活。

装饰器模式的应用

  • java io包
  • 解决对象之间多对多的组合关系
    【这点很重要！如果不理解这点，容易将装饰器模式与代理模式混淆】
